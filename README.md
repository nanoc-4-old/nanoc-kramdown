# nanoc-kramdown

This provides a [kramdown](http://kramdown.rubyforge.org/) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-kramdown`

## Usage

```ruby
filter :kramdown
```

Parameters passed to this filter will be passed on to kramdown.
